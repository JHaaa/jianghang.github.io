# Personal Academic Website

This is the personal academic website of Hang Jiang,
an incoming TTAP Assistant Professor at HKUST(GZ),
working broadly in OR/MS, AI Agents, FinTech,
mechanism design, and incentives.

## Design Philosophy

The website should look:

- clean
- modern
- minimal
- elegant
- research-oriented
- intellectually serious

Avoid:

- startup aesthetics
- excessive animations
- flashy gradients
- overly playful UI
- corporate SaaS style
- cluttered layouts

## Typography

Prioritize:

- readability
- whitespace
- professional academic appearance
- subtle prestige

## Coding Principles

- keep components modular
- avoid unnecessary dependencies
- preserve mobile responsiveness
- prefer simple and maintainable code
- avoid overengineering

## Website Goals

The website should help:

- communicate research clearly
- establish scholarly identity
- recruit future PhD students
- present a modern academic profile
- maintain long-term professionalism

## Content Policy

Do NOT substantially modify, rewrite, shorten,
or reinterpret academic content unless explicitly asked.

Focus primarily on:

- presentation
- layout
- typography
- spacing
- responsiveness
- visual polish
- UI/UX improvements

Preserve:

- research descriptions
- publication wording
- academic tone
- scholarly positioning
- technical terminology

When content edits are necessary,
prefer minimal and conservative changes.
